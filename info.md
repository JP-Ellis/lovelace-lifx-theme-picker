# LIFX Theme Picker

Browse the 42 built-in LIFX themes and apply one to a light or area with a
tap. Each theme is previewed live as a multi-point mesh gradient in the OkLCh
color space, so what you see is roughly what your bulbs will paint.

Add a card with:

```yaml
type: custom:lifx-theme-picker
entity: light.kitchen
```

See the full README on the [GitHub repo](https://github.com/JP-Ellis/lovelace-lifx-theme-picker)
for screenshots and configuration options.
